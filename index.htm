Introduction

The following project contained in this repository is about learning DevOps, Git, and Docker to work on these skills but It will focus mainly on Git.

DevOps is a combination of two words which are software development, and Operations. This allows a single team to handle the entire application lifecycle, from development to testing, deployment, and operations. DevOps helps you to reduce the disconnection between software developers, quality assurance(QA) engineers, and system administrators.

DevOps basics provides depth knowledge of various DevOps tools such as Git, Ansible, Docker, Puppet, Jenkins, Chef, Nagios, and Kubernetes.
